Radical Shop 2000

A 90s-themed e-commerce web application built with React and JSON Server.

How to Run:
Terminal 1 - Backend
cd Downloads/finalshop/backend
npm install
npm start
Runs on http://localhost:5000

Terminal 2 - Frontend
cd Downloads/finalshop/frontend
npm install
npm start
Runs on http://localhost:3000
Features

Browse products on the home page
Search products by name or category
Add items to your cart
Increase or decrease item quantity
Delete items from cart
Cart saves on page refresh

Components:

App.jsx - main component, handles all state and logic
NavBar.jsx - top navigation with search bar and cart button
SearchBar.jsx - search input component
Card.jsx - individual product card
ShoppingCart.jsx - cart page with quantity controls and order summary

Tech Stack:

React (useState, useEffect, props)
Bootstrap 5 for grid layout
Custom CSS for 90s styling
JSON Server for backend database
